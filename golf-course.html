3. SSE Transport (Legacy MCP)

Server-Sent Events

For mcp-remote or older MCP clients. Persistent connection.

# Connect
curl -N https://mcp.nexvigilant.com/sse

# Send message
curl -X POST https://mcp.nexvigilant.com/message \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,
       "method":"tools/list"}'

4. HTTP REST (Direct)

REST — any HTTP client

No MCP client needed. Standard HTTP for any framework.

# List all tools
curl https://mcp.nexvigilant.com/tools

# Get single tool schema
curl https://mcp.nexvigilant.com/tools/api_fda_gov_search_adverse_events

# Call a tool
curl -X POST https://mcp.nexvigilant.com/rpc \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,
       "method":"tools/call",
       "params":{"name":"api_fda_gov_search_adverse_events",
                 "arguments":{"drug":"semaglutide"}}}'

5. Python SDK

Python — pip install mcp

For custom agent frameworks, notebooks, and pipelines.

from mcp import ClientSession
from mcp.client.sse import sse_client

async with sse_client(
    "https://mcp.nexvigilant.com/sse"
) as (read, write):
    async with ClientSession(read, write) as session:
        await session.initialize()
        tools = await session.list_tools()
        result = await session.call_tool(
            "api_fda_gov_search_adverse_events",
            {"drug": "metformin"}
        )
